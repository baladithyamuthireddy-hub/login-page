# 🔐 Login & Signup Web Application

A modern and responsive Login & Signup web application built using HTML, CSS, and JavaScript.  
This project focuses on creating a clean authentication user interface with responsive layouts and interactive form validation.

---

# 🚀 Project Overview

This project is a beginner-friendly authentication UI application designed to practice frontend web development concepts like:
- Form design
- Responsive layouts
- Input validation
- Modern UI styling
- JavaScript interactivity

The interface is clean, user-friendly, and optimized for both desktop and mobile devices.

# ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/baladithyamuthireddy-hub/login-page.git
```

2. Open project folder

```bash
cd login-page
```

3. Run the project

Open `index.html` in your browser.
